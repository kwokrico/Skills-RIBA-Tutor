# Legal & Safety: Planning + Heritage (Listed Buildings / NPPF)

## Purpose
Use where scenarios involve converted mills, conservation areas, Grade II listings, heritage assets, or planning risk/programme implications.

## A. Statutory anchors (exam-facing summaries)
### Planning (Listed Buildings and Conservation Areas) Act 1990 — practical meaning
- Works affecting the character of a **listed building** typically require **Listed Building Consent (LBC)**.\n- Unauthorised works can create serious enforcement/prosecution risk.\n- Advice must be documented clearly to protect the practice.\n\n### NPPF — how to frame the argument
- Identify **significance** of the heritage asset.\n- Assess **harm** (including “less than substantial” vs substantial where relevant).\n- Set out **public benefits** and mitigation.\n\n## B. Process checklist (use as a short table if needed)
| Step | Purpose | Key evidence |
|---|---|---|
| Pre-app engagement | de-risk policy/consent path | meeting notes, constraints summary |
| Heritage statement | set significance + impact | statement, drawings, photos |
| LBC + planning strategy | correct consent sequencing | application pack, consultations |
| Conditions/discharge | programme + cost control | conditions tracker |

## C. Risk and mitigation prompts
- **Programme risk**: heritage consultation and redesign loops.\n- **Cost risk**: specialist materials/contractors; surveys.\n- **Enforcement risk**: client “doing works early”.\n- **Mitigation**: written advice, staged consents, clear scope, change control.\n\n## D. Output template
